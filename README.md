## Project Overview
This project is a **Spam Detection System** built using **Python, FastAPI, and React**. It classifies text messages into **spam** or **not spam** using machine learning techniques. The backend leverages **scikit-learn** for model training, while the frontend provides a simple **user interface** to test messages in real-time.

---

## Motivation
Spam messages are a common problem in digital communication, often containing scams, phishing attempts, or unwanted advertising. This project aims to **help users automatically detect and filter spam messages**, improving safety and productivity.

---

## Features
- Train a **Random Forest model** on a cleaned dataset of labeled messages.  
- Save trained **model** and **vectorizer** for inference.  
- FastAPI backend exposes endpoints for:
  - Single message prediction
  - Batch predictions  
- React frontend allows easy testing of messages.  
- Fully functional local deployment.

---

## Technologies Used
- **Python:** pandas, scikit-learn, FastAPI, Uvicorn  
- **Frontend:** React.js  
- **Data:** CSV dataset with `label` and `text` columns  
- **Version Control:** Git & GitHub
